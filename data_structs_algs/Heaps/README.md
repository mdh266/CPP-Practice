# Heap Data Structure
A min heap written in C+11 using template classes and a heap abstract base class.

## Requirements
- C++11
- CMake 3.0+

## Compiling
From top directory (this one) :
	
<code> mkdir build </code>

<code> cd build </code>

<code> cmake .. </code>

<code> make </code>

## Running 
From /build/ directory type:

<code> ./main </code>
