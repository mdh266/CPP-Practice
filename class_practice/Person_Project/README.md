## Person Project
An example code on how to write classes that use inheritence and
polymorphism.

## Requirements
- CMake 2.7 +

## Compiling
From top directory (this one) :
	
<code> mkdir build </code>

<code> cd build </code>

<code> cmake .. </code>

<code> make </code>

## Running 
From /build/ directory type:

<code> ./main </code>
